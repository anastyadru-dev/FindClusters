Параметры:

1. M (Ширина сетки):
   – Описание: Количество колонок в игровом поле.
   – Настройка: Должен быть настраиваемым параметром через UI.

2. N (Высота сетки):
   – Описание: Количество строк в игровом поле.
   – Настройка: Должен быть настраиваемым параметром через UI.

3. X (Количество типов иконок):
   – Описание: Число различных типов иконок, используемых для заполнения поля.
   – Настройка: Должен быть настраиваемым параметром через UI.

4. Y (Минимальный размер кластера):
   – Описание: Минимальное количество соединённых иконок одного типа, необходимое для формирования кластера.
   – Настройка: Должен быть настраиваемым параметром через UI.

Ограничения:

• Все параметры (M, N, X, Y) должны быть видимыми и изменяемыми в интерфейсе игры без необходимости редактирования кода.
• Игра должна корректно работать при разумных значениях параметров, указанных выше.
• При нажатии кнопки "Start" поле должно очищаться, заполняться случайными иконками из набора X, а затем автоматически находиться и подсвечиваться все кластеры размером ≥ Y.
• Подсветка кластеров должна быть явно заметной и однозначно идентифицировать ячейки, входящие в один кластер.
• Код должен быть структурирован, отделяя логику генерации, поиска кластеров и отображения.
• Не должно быть ошибок в консоли во время выполнения игры.
